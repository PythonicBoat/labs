# OS Lab Notes

mkdir - make directory 
```sh
mkdir <dir_name>
```
ls - show sub-folders and file 
```sh
ls <-A>
```
sh - to evalute a shell expression
```md
sh <file.sh> <expression>
```
pwd - path to directory
```sh
pwd
```
cd - change directory
```sh
cd <dir_name>
```
cat - show content of file
```md
cat <file_name>
```
date - show date
```md
date <format> <time_zone>
```
